# Extension Methods

## AddNetMediate()

Registers all NetMediate services. Generated by the source generator.

```csharp
builder.Services.AddNetMediate();
```

## UseNetMediate()

Manual registration method (advanced usage).

```csharp
builder.Services.UseNetMediate(configure =>
{
    configure.RegisterCommandHandler<MyHandler, MyCommand>();
});
```
